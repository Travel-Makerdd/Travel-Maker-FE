# 초기 세팅

- 프로젝트를 클론 받습니다.
- 주의! npm 이 아니고 pnpm 을 사용합니다. `pnpm install` 로 의존성을 설치합니다.
- 개발 서버를 띄울 때는 `pnpm dev` 를 사용합니다. http://localhost:3000 에서 확인할 수 있습니다.

# 기술 스택

- Next.js
- Tailwind CSS (스타일링)
- TypeScript & Javascript

# Shadcn UI 활용법

- `pnpm dlx shadcn@latest add button` 형태로 기본 ui 컴포넌트를 추가합니다.
